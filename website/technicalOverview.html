Technical Overview

Our site allows users to post listings for furniture or other items they want to sell and buy other user's items. If they lack transportation or are out of state at the time of purchase, they can request a driver on the site.

This site is intended to be used by SMU students or recent graduates. We limited use to SMU to create more trust since this service involves face to face interaction.

Feature Set

  • Ability to post a listing for any logged in user to see
  • Search functionality with filters that allows users to find specific listings
  • Driver request system
  • Driver signup system

Implementation

Our site was implemented with a MySQL backend, PHP middleware and JavaScript/HTML front end. We chose to use JQuery as our JavaScript framework and used the Sass CSS precompiler for some of our styling. We created an Android application that offers slimmed down, lightweight functionality.

API Documentation

The two main APIs used in this site are Cloudinary and MailBoxLayer provided by APILayer. Cloudinary is used for image storage, resizing and quick thumbnail access. MainBoxLayer used for email verification to make sure emails are of the correct domain and are a valid address.
